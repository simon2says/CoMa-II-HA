        \begin{enumerate}[label=(\alph*)]
            \item \begin{theorem*}
                $c_1 = c_2 = \frac{1}{2}$.
            \end{theorem*}
            \begin{proof}[Beweis durch vollstaendige Induktion.]
                Wir zeigen durch vollstaendige Induktion, dass in jeder $(i + 1)$-ten Iteration der While-Schleife
                \[
                    h'(k, i) = \Big(h(k) + \frac{1}{2}i^2 + \frac{1}{2}i\Big) \mod m
                \]
                ist.
                
                Induktionsanfang $(i = 0)$: $j$ ist die Position, an der nach dem Eintrag geschaut wird. Damit ist $h(k)$ allgemein $j$. In der ersten Iteration ist 
                \[
                    j = h'(k, 0) = \Big(h(k) + \frac{1}{2}\cdot 0^2 + \frac{1}{2}\cdot 0\Big) \mod m
                \]
                und der Induktionsanfang gilt.

                Induktionsvoraussetzung: Fuer ein beliebiges, aber feste Iteration $i + 1$ gelte:
                \[
                    j = h'(k, i) \Big(h(k) + \frac{1}{2}i^2 + \frac{1}{2}i\Big)\mod m
                \]
                Induktionsschritt: Dann gilt auch:
                \[
                    j = h'(k, i + 1) = \Big(h(k) + \frac{1}{2}(i + 1)^2 + \frac{1}{2}(i + 1)\Big)\mod m.
                \]
                In jeder Iteration der While-Schleife wird j + $i =$ Schleifenaufrufe + 1 gerechnet. Daher gilt im $i$-ten Schleifenaufruf laut Induktionsvoraussetzung:
                \begin{align*}
                    j = h'(k, i + 1) &= \Big(h(k) + \frac{1}{2}i^2 + \frac{1}{2}i + i + 1\Big)\mod m\\
                    &= \Big(h(k) + \frac{1}{2}i^2 + \frac{3}{2}i + 1\Big)\mod m\\
                    &= \Big(h(k) + \frac{1}{2}i^2 + \frac{1}{2}i + i + \frac{1}{2} + \frac{1}{2}\Big) \mod m\\
                    &= \Big(h(k) + \frac{1}{2}(i^2 + 2i + 1) + \frac{1}{2}(i + 1)\Big)\mod m\\
                    &= \Big(h(k) + \frac{1}{2}(i + 1)^2 + \frac{1}{2}(i + 1)\Big)\mod m
                \end{align*}
                Gilt.
            \end{proof}
            \item 
        \end{enumerate}
